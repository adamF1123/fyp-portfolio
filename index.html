REKEK – E‑commerce (Final Year Project)

A simple but complete badminton e‑commerce website built with PHP + MySQL. It keeps the original design and focuses on clean flows: browse products → add to cart → checkout → shipment tracking → returns & refunds. Admin pages include basic dashboards and product management.

Main Modules

  • Product catalogue with categories (rackets, shoes, shirts, shuttlecocks).
  • Cart & Checkout (quantity update, remove, total calculation).
  • Orders & Shipments (simple status: pending → shipped → delivered).
  • Returns/Refunds (user request + admin approve/reject + reply message).
  • Admin: add/edit products, see top sellers, basic KPIs.
